## Upgrade Instructions

Follow these steps to upgrade to the latest version seamlessly.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full export of your strategies.

````bash
npx mps-cli@latest export --project-id your-project-id --output ./backup
````

  </Step>
  <Step title="Update Package" icon="download">

    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn">
````bash
npm install mps@latest
````
````bash
yarn add mps@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute any required database migrations.

````bash
npx mps-cli migrate --env production
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test core features like dashboard access and collaboration.

  </Step>
</Steps>

<Callout kind="alert">
  For `breaking` changes in v1.0.0, review your API calls to `/v1/strategies` endpoints.
</Callout>
