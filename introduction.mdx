## Overview

Master in Public Strategy empowers you to organize, manage, and collaborate on project documentation seamlessly. You create structured spaces for your teams to build, version, and share knowledge without the hassle of scattered files or complex tools. This platform streamlines your workflow, ensuring all critical information stays accessible and up-to-date.

  Primary brand color: <code>#3B82F6</code>. Use it for buttons, links, and accents to maintain visual consistency.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit <code>https://dashboard.example.com/signup</code> and create your account with email or OAuth.
  </Step>
  <Step title="Create a Space" icon="folder-plus">
    Click "New Space" and name it for your project, like "Public Strategy Docs".
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Start with a blank page or import Markdown. Use the editor for rich content.
  </Step>
  <Step title="Invite Collaborators" icon="mail">
    Share the space link or add emails directly.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  // Fetch your spaces via API
  const response = await fetch('https://api.example.com/v1/spaces', {
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`
    }
  });
  const spaces = await response.json();
  console.log(spaces);
  ```
  ```bash
  curl -H "Authorization: Bearer YOUR_API_KEY" \
       https://api.example.com/v1/spaces
  ```
</CodeGroup>
